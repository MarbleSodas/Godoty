Godoty - AI Assistant for Godot Game Engine
Version: 1.0.0
Platform: macOS (Apple Silicon / Intel)

INSTALLATION:
1. Copy Godoty.app to /Applications/
2. Right-click and select "Open" (first time only, due to Gatekeeper)
3. The app will create ~/.godoty/ folder on first run
4. Open the app and configure your OpenRouter API key in the settings panel

REQUIREMENTS:
- macOS 10.15 or later
- OpenRouter API key (get from https://openrouter.ai/)

TROUBLESHOOTING:
- If the app won't open due to security settings:
  Open Terminal and run:
  xattr -cr /Applications/Godoty.app

- If you see permission errors:
  Check ~/.godoty/ folder permissions

USAGE:
- Launch Godoty.app
- The app will open a window with the Godoty interface
- Connect to your Godot project using the Godot plugin
- Use the AI assistant to help with your game development

For more information, visit: https://github.com/your-repo/godoty
