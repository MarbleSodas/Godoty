Godoty - AI Assistant for Godot Game Engine
Version: 1.0.0
Platform: Windows 10/11

INSTALLATION:
1. Copy Godoty.exe to a folder (e.g., C:\Program Files\Godoty\)
2. Run Godoty.exe
3. The app will create %USERPROFILE%\.godoty\ folder on first run
4. Open the app and configure your OpenRouter API key in the settings panel

REQUIREMENTS:
- Windows 10 or later
- .NET Framework 4.7.2+ (usually pre-installed)
- OpenRouter API key (get from https://openrouter.ai/)

TROUBLESHOOTING:
- If Windows Defender blocks the app:
  Click "More info" > "Run anyway"

- If you see "VCRUNTIME140.dll missing":
  Install Visual C++ Redistributable from Microsoft

- If the app crashes on startup:
  Check %USERPROFILE%\.godoty\ folder for error logs

USAGE:
- Run Godoty.exe
- The app will open a window with the Godoty interface
- Connect to your Godot project using the Godot plugin
- Use the AI assistant to help with your game development

For more information, visit: https://github.com/your-repo/godoty
